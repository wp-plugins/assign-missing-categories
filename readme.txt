=== Assign Missing Categories ===Contributors: sillybeanTags: categoriesRequires at least: 2.3Tested up to: 2.7.1Stable tag: 1.0Assigns the default category to all posts that have no category after you've converted your categories to tags.== Description ==If you have converted categories to tags and are now left with a lot of posts that have no category assigned (that show up with 'Uncategorized,' unlinked, as the category under Manage Posts), you can use this plugin to assign the default category to all those posts at once.== Installation ==Installation is pretty standard:e.g.1. Upload the files to the `/wp-content/plugins/` directory1. Activate the plugin through the 'Plugins' menu in WordPress
